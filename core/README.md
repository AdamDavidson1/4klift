4klift Core Files
=======================

The following files are copied into a 4klift file structure after install.

- 4klift.php -> core/
- cli.php -> core/
- index.php -> web/
- Vagrantfile -> ../
- ./cfg/example-application.yml -> cfg/
- ./cfg/example-defaults.yml -> cfg/
- ./cfg/example-parameters.yml -> cfg/
- ./dev -> ../dev
